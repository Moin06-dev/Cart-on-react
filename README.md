# React Cart Page 🛒

This project demonstrates a simple Cart page built using React.

### Prerequisites

Make sure you have Node.js and npm installed. You can download it from [nodejs.org](https://nodejs.org/).

### Installing Dependencies

```bash
npm install
```

### Running the App

```bash
npm start
```

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser. 🌐

The page will reload when you make changes.\
You may also see any lint errors in the console. 🛠️

## Features

- Add items to the cart 🛍️
- Remove items from the cart ❌
- Update quantities 🔢
- Calculate total price 💲

## Available Scripts
